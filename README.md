# React Typescript Tetris App


## Some of technoligies used
- Typescript
- Vite
- React Hooks
- styled-components


## Usage
```bash
# Install dependencies and run the app

 $ npm install

 $ npm run dev

# Build app for production (gets output in the 'dist' directory

 $ npm run build
```
